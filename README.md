### Fonctionnalités détaillées

- Barre de recherche
- Filtres par catégorie
- Tri
- Mode sombre
- Pages par plateforme
## Démarrer en local

Option rapide (macOS):
```bash
open index.html
```

Serveur local recommandé (meilleure compatibilité):
```bash
python -m http.server 8000
# Ouvrir ensuite http://localhost:8000
```
